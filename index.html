Le développement en Java impose certaines contraintes : nécessité de maitriser de nombreux concepts généraux (notamment celui de la POO, les design patterns, etc ...) et spécifiques aux plate-formes Java (standard, entreprise et mobile) selon les besoins nécessité de maitriser de nombreuses API qui sont de bas niveau quasi-absence d'outils de type RAD.
